## Open Source

### [phantom-detect](https://github.com/brianrutherford/phantom-detect)
Detect steganographic covert channels in LLM outputs. Statistical anomaly detection, behavioral baselining, and cross-agent correlation analysis.
- **Language:** Python
- **Status:** Alpha (v0.1.0)
- **Install:** `pip install phantom-detect`

### pqc-py *(coming soon)*
Post-quantum cryptography for Python via Rust. FIPS 203/204/205 (Kyber, Dilithium, SPHINCS+) with ergonomic Python bindings.
- **Language:** Rust + Python (PyO3)
- **Status:** In development

### hndl-detect *(coming soon)*
Detect Harvest-Now-Decrypt-Later attack patterns in network telemetry.
- **Language:** Python
- **Status:** In development

### behavioral-entropy *(coming soon)*
AI agent behavioral fingerprinting library. Compute entropy signatures from agent behavior patterns for authentication and anomaly detection.
- **Language:** Python
- **Status:** In development
