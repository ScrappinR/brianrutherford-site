## Open Source

### [phantom-detect](https://github.com/ScrappinR/phantom-detect)
Detect steganographic covert channels in LLM outputs. Statistical anomaly detection, behavioral baselining, and cross-agent correlation analysis.
- **Language:** Python
- **Status:** Alpha (v0.1.0)
- **Install:** `pip install phantom-detect`

### [pqc-py](https://github.com/ScrappinR/pqc-py)
Post-quantum cryptography for Python via Rust. FIPS 203/204/205 (Kyber, Dilithium, SPHINCS+) with ergonomic Python bindings and automatic key zeroization.
- **Language:** Rust + Python (PyO3)
- **Status:** Alpha (v0.1.0)
- **Install:** `pip install pqc-py` (requires Rust toolchain for build)

### [hndl-detect](https://github.com/ScrappinR/hndl-detect)
Detect Harvest-Now-Decrypt-Later attack patterns in network telemetry. Six signal types, APT attribution, quantum vulnerability assessment.
- **Language:** Python
- **Status:** Alpha (v0.1.0)
- **Install:** `pip install hndl-detect`

### [behavioral-entropy](https://github.com/ScrappinR/behavioral-entropy)
AI agent behavioral fingerprinting library. Compute Shannon entropy signatures from agent behavior patterns for authentication and anomaly detection.
- **Language:** Python
- **Status:** Alpha (v0.1.0)
- **Install:** `pip install behavioral-entropy`
